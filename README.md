# tw-post-query

CLI tool to query Taiwan Post (Mail) status.


## Usage

```sh
tw-pq -k <package-tracking-serial>
```

## System Prerequisite

- [curl](https://curl.haxx.se/)
- [jq](https://stedolan.github.io/jq/)

## Reference

- [中華郵政 - 國內快捷/掛號/包裹查詢](http://postserv.post.gov.tw/pstmail/main_mail.html)
